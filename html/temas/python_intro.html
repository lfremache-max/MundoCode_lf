Maestro Python

Sintaxis limpia y poderosa.

1. Imprimir
Usa la función print('Texto'). Es la forma más básica de mostrar información en la consola.
2. Comentarios
Todo lo que escribas después de un símbolo # será ignorado por Python. Úsalo para notas.
3. Extensión
Los archivos de código fuente Python deben guardarse con la extensión .py para ejecutarse.
4. Variables
No necesitas declarar el tipo (int, string). Solo asigna el valor: x = 5 y Python entiende qué es.
5. Indentación
Vital en Python. Usa sangría (espacios) para definir qué código está dentro de un bucle o función.
6. Listas
Colecciones ordenadas y modificables. Se crean con corchetes cuadrados: lista = [1, "texto", 3.5].
7. Comparación
Un igual = asigna. Doble igual == compara igualdad. El operador != es "diferente de".
8. Bucle For
Recorre listas fácilmente: for item in lista:. No usa contadores complejos como C o Java.
9. Funciones
Se definen con la palabra clave def. Ejemplo: def saludar(): print("Hola").
10. Mayúsculas
Python es "Case Sensitive". La variable Edad es diferente de edad. Ten cuidado.
