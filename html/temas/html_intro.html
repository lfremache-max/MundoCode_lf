Introducción a HTML

HTML (HyperText Markup Language) es el bloque de construcción más básico de la Web.


Estructura básica

Toda página HTML comienza con una estructura de árbol:

<!DOCTYPE html>
<html>
  <head>
    <title>Título</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
  </body>
</html>
