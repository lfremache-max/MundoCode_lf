Arquitecto HTML5

La estructura fundamental de la web.

1. Etiquetas
HTML usa "tags" o etiquetas rodeadas por corchetes angulares. La mayoría tienen apertura <p> y cierre </p>.
2. Estructura
Todo documento requiere: <html>, <head> (metadatos) y <body> (contenido visible).
3. Títulos
Van desde <h1> (el más importante) hasta <h6>. Los buscadores los usan para entender tu página.
4. Imágenes
La etiqueta <img> no tiene cierre. Usa el atributo src para la ruta y alt para la descripción.
5. Enlaces
Se crean con la etiqueta <a>. El atributo vital es href="..." que indica el destino.
6. Listas
Usa <ul> para listas con viñetas (desordenadas) y <ol> para numeradas. Los ítems son <li>.
7. Párrafos
La etiqueta <p> define bloques de texto. Los navegadores añaden márgenes automáticos entre ellos.
8. Atributos
Proporcionan info extra. Van en la etiqueta de apertura. Ejemplo: class="texto-rojo" o id="menu".
9. Doctype
<!DOCTYPE html> debe ser la primera línea. Le dice al navegador que usas HTML5 moderno.
10. Comentarios
Se escriben así: <!-- Texto -->. Sirven para dejar notas que no se ven en la página web.
