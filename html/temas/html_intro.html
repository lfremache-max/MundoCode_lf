Domina la estructura web.

1. Significado
HTML (HyperText Markup Language) es el esqueleto de la web. Define la estructura semántica, no el diseño visual.
2. Estructura
Sigue un orden estricto: <html> envuelve todo, <head> configura metadatos y <body> muestra el contenido.
3. Títulos
Van de <h1> a <h6>. El h1 es el título principal y único por página, vital para el SEO y la jerarquía.
4. Párrafos
La etiqueta <p> agrupa bloques de texto. Los navegadores añaden márgenes automáticos para separarlos visualmente.
5. Imágenes
Etiqueta de cierre automático: <img src="...">. El atributo 'alt' es obligatorio para accesibilidad.
6. Enlaces
La etiqueta <a href="..."> conecta páginas. Usa target="_blank" para abrir en una pestaña nueva.
7. Listas
<ul> crea listas con puntos (bullets) y <ol> listas numéricas. Los ítems van en <li>.
8. Salto Línea
<br> fuerza un salto de línea sin crear un nuevo párrafo. Es útil para direcciones o poemas.
9. Video
<video src="..." controls> inserta multimedia nativa con controles de play y pausa integrados.
10. Doctype
<!DOCTYPE html> es la primera línea obligatoria. Le dice al navegador que use el modo estándar de HTML5.
