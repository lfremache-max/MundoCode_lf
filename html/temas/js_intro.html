Lógica JavaScript

Programación dinámica.

1. Etiqueta
Todo el código JavaScript debe ir dentro de las etiquetas <script> para ser ejecutado.
2. Externo
Para archivos separados, usa el atributo src: <script src="logica.js">.
3. Alertas
alert("Texto"); crea una ventana emergente en el navegador que detiene la ejecución hasta que se cierra.
4. Crear Función
Se definen con function nombre() {}. Son bloques de código reutilizables.
5. Ejecutar
Para que una función actúe, debes "llamarla" usando su nombre y paréntesis: nombre().
6. Condicional IF
La condición va entre paréntesis. Ejemplo: if (edad > 18) { ... }.
7. Bucle While
Repite un bloque de código mientras una condición sea verdadera: while (i < 10).
8. Comentarios
// para una sola línea y /* ... */ para bloques de texto grandes.
9. Constantes
Usa const para valores que no cambiarán y let para variables que sí cambiarán. Evita 'var'.
10. Asignación
Un igual = asigna valor. Dos o tres iguales === comparan si dos valores son idénticos.
