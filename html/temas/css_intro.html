Introducción a CSS

CSS (Cascading Style Sheets) es el lenguaje que usamos para dar estilo a un documento HTML. Describe cómo se deben mostrar los elementos.


Sintaxis Básica

Una regla CSS consiste en un selector y un bloque de declaración:

h1 {
  color: blue;
  font-size: 12px;
}
  • Selector: Apunta al elemento HTML que quieres estilizar (ej. h1).
  • Propiedad: Lo que quieres cambiar (ej. color).
  • Valor: El ajuste específico (ej. blue).
