Estilista CSS3

Dale vida y color a la web.

1. Significado
CSS (Cascading Style Sheets) separa el diseño del contenido. Permite crear layouts y adaptar la web a móviles.
2. Etiqueta Link
Usa <link rel="stylesheet" href="...">. Es el método estándar para conectar hojas de estilo externas.
3. Ubicación
El enlace CSS siempre debe ir dentro del <head> para cargar los estilos antes de mostrar la página.
4. Fondo
background-color define colores planos, mientras que background-image permite usar fotos.
5. Selector ID
Se selecciona con # (ej: #menu). Debe ser único en toda la página y tiene alta prioridad.
6. Selector Clase
Se selecciona con punto . (ej: .boton). Son reutilizables para múltiples elementos.
7. Color Texto
La propiedad es color (no font-color). Acepta nombres, hexadecimales y RGB.
8. Comentarios
Se escriben así: /* ... */. El navegador los ignora, sirven para organizar tu código.
9. Tamaño Fuente
Controlado por font-size. Usa unidades como 'px' (fijo) o 'rem' (relativo y accesible).
10. Visibilidad
visibility: hidden oculta el objeto pero deja el hueco vacío. display: none lo elimina por completo.
