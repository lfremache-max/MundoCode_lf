Experto SQL

Bases de datos dominadas.

1. Significado
SQL (Structured Query Language) es el estándar para gestionar bases de datos relacionales, permitiendo guardar y leer datos.
2. Consultar
SELECT * FROM tabla trae todo, pero se recomienda especificar columnas (ej: SELECT nombre...) para mayor rapidez.
3. Todo
El asterisco * actúa como comodín para decir "todas las columnas". Úsalo solo cuando realmente necesites todo.
4. Filtrar
Usa WHERE para condiciones. Ejemplo: WHERE edad > 18. Puedes combinar con AND/OR.
5. Ordenar
ORDER BY columna ordena los resultados. Añade DESC al final para orden descendente (Z-A).
6. Borrar
DELETE FROM tabla WHERE id=1. ¡Cuidado! Si olvidas el WHERE, borrarás todos los datos de la tabla.
7. Insertar
INSERT INTO tabla VALUES (...). Asegúrate de que los valores sigan el orden exacto de las columnas.
8. Modificar
UPDATE tabla SET campo=valor actualiza datos. Al igual que con Delete, ¡no olvides el WHERE!
9. Únicos
SELECT DISTINCT columna devuelve solo valores únicos, eliminando automáticamente los duplicados.
10. Patrones
LIKE busca coincidencias parciales. Ej: 'a%' busca todo lo que empiece por la letra 'a'.
